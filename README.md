# FidgetKey

Tapping keys on your numpad lights up the corresponding shape on screen. Release the key and it disappears. Meant as a fidget toy to relax.

## Requirements

- macOS 14.0+
- Keychron K0 Max (or any numpad with standard layout)

## Layout

```
[Home] [End ] [PgUp] [PgDn]
[NmLk] [ / ] [ * ] [ - ]
[ 7 ] [ 8 ] [ 9 ] [     ]
[ 4 ] [ 5 ] [ 6 ] [  +  ]
[ 1 ] [ 2 ] [ 3 ] [     ]
[   0   ] [ . ] [Enter]
```

Simultaneous key presses are supported.
